# GSharp
[日本語説明はこちらへ](README.JP.md)  
[Click here for English Description](README.EN.md)

.NET Framework 코드를 시각적으로 작성할 수 있는 통합 프레임워크 입니다.  
본 프레임워크를 활용하여 비주얼 프로그래밍 솔루션등의 개발이 가능합니다.
